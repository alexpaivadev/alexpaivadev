## 🏗️ O que eu construo no dia a dia

### 📊 Data Warehouse & ETL
- Projetei e mantenho um **Data Warehouse dimensional** (PostgreSQL 14) com tabelas fato e dimensão para boletos, veículos, consultores, cotações CRM e calendário operacional
- Pipelines ETL automatizados que ingerem dados das **APIs Hinova e PowerCRM**, com controle de ingestão, alertas de falha e monitoramento de saúde via WhatsApp
- Modelagem dimensional com regras de negócio específicas (dias úteis seg–sáb, calendário de feriados, janelas de inadimplência)

### 📈 BI & Dashboards
- **Power BI**: dashboards de carteira, movimentação, inadimplência e desempenho comercial conectados diretamente ao DW
- Migração e padronização de medidas DAX, criação de dimensões compartilhadas (`dim_calendario`, `dim_consultores`)
- Relatórios executivos para a rede de cooperativas com análise de receita, descontos e segmentação por tipo de veículo

### ⚡ Automação & Integrações
- **n8n** como orquestrador central: workflows de ETL, notificações, processamento de dados e integrações entre sistemas
- Automação de envio de boletos via **WhatsApp** (Evolution API + Chatwoot) com regras de cobrança multi-estágio e deduplicação
- Pipeline inteligente: Trello → GPT-4o-mini → Notion → WhatsApp para avaliação automatizada de reparos veiculares

### 🖥️ Infraestrutura Self-Hosted
- **Docker Swarm** em VPS com stack completa: Traefik (reverse proxy + SSL), PostgreSQL, n8n, Supabase (13 serviços), Portainer, Evolution API, Chatwoot
- Backups automatizados com `pg_dump` + rotação de 7 dias, logrotate, monitoramento de saúde dos serviços
- Subdomínios gerenciados sob `alexpaiva.dev` com arquitetura de isolamento entre serviços
